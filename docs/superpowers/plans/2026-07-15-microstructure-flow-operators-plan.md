# Microstructure & Order-Flow Operators (Tranche 1a) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add the flow-foundation microstructure operators (order-flow imbalance, signed volume, tick-rule sign, Kyle's lambda, Amihud illiquidity) to the screamer core as first-class, discoverable, documented Python operators.

**Architecture:** A new Python module `screamer/microstructure.py` holds the operators. Each is a thin class that **composes existing C++ operators** (or aliases one), so causality and batch==stream are inherited from the C++ engine rather than re-implemented. The module is wired into the codegen (`generate_screamer__init__.py`), the docs/help pipeline (`docs/functions_micro/*.md` -> `help.json`), and the doc-coverage guardrail, exactly as `streams.py`/`dag.py` are.

**Tech Stack:** Python 3.11, numpy, the screamer C++ bindings (`RollingBeta`, `EwBeta`, `RollingMean`, `Diff`, `Sign`, `Abs`, `Div`, `Ffill`), pytest.

## Global Constraints

```python
"""Microstructure and order-flow operators.

Causal streaming operators for order flow, price impact, and liquidity. Each one
either composes existing screamer operators or aliases one, so causality and the
batch == stream guarantee are inherited from the engine. Popular models are
exposed under their canonical name with teaching-quality docs (see
docs/functions_micro/).
"""
import numpy as np

__all__ = ["OFI"]


class OFI:
    """Order-flow imbalance: (buy_volume - sell_volume) / (buy_volume + sell_volume).

    Cont-Kukanov-Stoikov style normalized signed flow, in [-1, 1]; 0 on an empty
    bucket. Stateless and elementwise, so it is trivially causal.
    """

    def __call__(self, buy_volume, sell_volume):
        buy = np.asarray(buy_volume, dtype=float)
        sell = np.asarray(sell_volume, dtype=float)
        total = buy + sell
        safe = np.where(total > 0, total, 1.0)
        return np.where(total > 0, (buy - sell) / safe, 0.0)
```

```python
from . import Diff, Sign, Abs, Div, Ffill


class SignedVolume:
    """Signed order flow: sign * volume (aggressor-signed volume)."""

    def __call__(self, sign, volume):
        return np.asarray(sign, dtype=float) * np.asarray(volume, dtype=float)


class TickRuleSign:
    """Trade sign by the tick rule: +1 on an up-tick, -1 on a down-tick, and the
    previous sign carried forward when the price is unchanged (first bar 0).

    Composed as Ffill(sign(diff) / |sign(diff)|): the division maps an unchanged
    tick (0/0) to NaN, and Ffill carries the last known +/-1 across it. Built on
    causal C++ operators, so it is causal and batch == stream by construction.
    """

    def __call__(self, price):
        price = np.asarray(price, dtype=float)
        d = Sign()(Diff(1)(price))          # -1 / 0 / +1, first is 0
        signed = Div()(d, Abs()(d))         # 0/0 -> NaN on unchanged ticks
        return Ffill()(signed)              # carry last +/-1 forward
```

Note: `from . import ...` inside the module is safe because these names are C++
bindings already imported by `screamer/__init__.py` before `microstructure` is
imported. If a circular-import error appears, switch to
`from .screamer_bindings import Diff, Sign, Abs, Div, Ffill`.

```python
def test_kyle_lambda_equals_rolling_beta():
    from screamer import RollingBeta
    from screamer.microstructure import RollingKyleLambda
    rng = np.random.default_rng(0)
    flow = rng.normal(size=200)
    ret = 2.5 * flow + rng.normal(scale=0.1, size=200)   # true impact slope 2.5
```

```python
class RollingKyleLambda:
    """Kyle's lambda over a trailing window: the price-impact / illiquidity slope
    of return on signed order flow (Kyle 1985). Specializes RollingBeta.
    """

    def __init__(self, window_size=20, start_policy="strict"):
        self._beta = RollingBeta(window_size, start_policy)

    def __call__(self, signed_flow, return_):
        return self._beta(return_, signed_flow)   # slope of return on flow


class EwKyleLambda:
    """Kyle's lambda, exponentially weighted (recursive). Specializes EwBeta."""

    def __init__(self, span=20.0):
        self._beta = EwBeta(span=span)

    def __call__(self, signed_flow, return_):
        return self._beta(return_, signed_flow)
```

```python
class AmihudIlliquidity:
    """Amihud (2002) illiquidity: rolling mean of |return| / notional. Large
    values mean price moves a lot per dollar traded (an illiquid, high-impact
    regime). A robust, cheap cousin of Kyle's lambda.
    """

    def __init__(self, window_size=20, start_policy="strict"):
        self._mean = RollingMean(window_size, start_policy)

    def __call__(self, return_, notional):
        ret = np.asarray(return_, dtype=float)
        notl = np.asarray(notional, dtype=float)
        return self._mean(np.abs(ret) / notl)
```
