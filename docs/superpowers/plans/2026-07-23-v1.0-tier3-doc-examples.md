## Canonical template (plotly), verbatim from `docs/functions_rolling/RollingMean.md`

````markdown
## Examples

### Usage example

```{eval-rst}
.. plotly::
    :include-source: True

    import numpy as np
    import plotly.graph_objects as go
    from screamer import RollingMean

    rng = np.random.default_rng(0)
    N = 300
    data = np.cumsum(rng.standard_normal(N))

    fig = go.Figure()
    fig.add_trace(go.Scatter(y=data, mode='lines', name='Input'))
    fig.add_trace(go.Scatter(y=RollingMean(30)(data), mode='lines', name='RollingMean(30)', line=dict(color='red')))
    fig.update_layout(
        title="Rolling mean, window 30",
        xaxis_title="Index", yaxis_title="Value",
        margin=dict(l=20, r=20, t=80, b=20),
        legend=dict(orientation="h", yanchor="bottom", y=1.02, xanchor="right", x=1),
    )
    fig.show()
```
````

For a stat whose scale differs from the input (dispersion, oscillators, slope, beta), use `make_subplots(rows=2, cols=1, shared_xaxes=True)` and put the operator output in the lower panel. Reference `docs/functions_micro/AmihudIlliquidity.md` for the two-panel idiom.
